# Safety and Non-goals

## Non-goals

- trading execution
- wallet integration
- private key handling
- auto-buy / auto-sell
- Telegram live sending
- production cron changes
- modifying existing Hermes / Fast DEX logic

## Guardrails

- Always include `not_a_buy_signal` in research notes.
- Missing data must be visible.
- Weak Mantle relevance must be explicit.
- Human decision is required for escalation.
