# Challenge Submission Summary

## Project name

Mantle Signal-to-Thesis Console

## Track

Research Agent / AI workflow / dashboard.

## Summary

Mantle Signal-to-Thesis Console is a self-hosted AI research dashboard that turns fragmented Web3 scanner outputs into structured ecosystem intelligence: evidence, risk flags, Mantle relevance, generated thesis, and human-confirmed decisions.

## What it demonstrates

- repeatable agent workflow
- typed signal schema
- append-only audit trail
- risk-first research notes
- dashboard for operators and judges
- no private keys or execution

## Why it is not just an AI wrapper

The core artifact is not a chat response. It is an auditable data workflow:

- validated events
- durable store
- materialized state
- deterministic scoring
- research note generation
- decision trail

## Mantle relevance

Signals are classified by Mantle relevance:

- Mantle native
- RWA distribution
- tokenized assets
- prediction markets
- AI-agent finance
- DeFi liquidity
- ecosystem scouting
- not relevant

## Safety

This is not a buy/sell system. It is read-only and watch-only.
